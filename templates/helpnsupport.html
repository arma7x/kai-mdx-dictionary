# Scanning MDX on high capacity storage take a long time to finish. Please 'Kill App' if you think the app was hang
# You can download MDX file from any forum
# Troubleshooting
  • To fix storage permission, goto Settings > Privacy & Security > App Permissions > MDX Dictionary > Memory Card Storage, then select Grant
  • Open MDX Dictionary, goto Menu > Reload Library or restart your device to reload all MDX files from storage
