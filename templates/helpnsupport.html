# Scanning files on high capacity storage take a long time to finish. Please 'Kill App' if you think the app was hang
# The '.mdx' and '.css' files (optional) must be in the same folder. Both files must use the same name
# This app does not provide MDX file. You can download those files here:
  • https://downloads.freemdict.com/
  • https://t.me/MDict_Dictionary
# Troubleshooting
  • To fix storage permission, goto Settings > Privacy & Security > App Permissions > MDX Dictionary > Memory Card Storage, then select Grant
  • To reload mdx files, goto Menu > Reload Library
  • Please restart your device if the app fail to open MDX file
# Shortcut key
  • Arrow Up or Arrow Down to scroll the page
  • Arrow Left or Arrow Right to jump between anchor tag
  • Enter to select anchor tag
